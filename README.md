# Project4

Yellevate has been struggling with client disputes. This has led to an approximate 5% annual loss of revenues (in USD). Using data, the company collected about these disputes. Identified the problems facing the business and came up with objectives that can solve them with data analysis. 路 Set data analysis goals according to these business objectives. 路 Loaded a .csv file in SQL to perform the necessary data cleaning. 路 Analyzed and visualize the data using Excel. 路 Generated insights from the analysis to provide recommendations on probable strategies to deal with disputes effectively.
